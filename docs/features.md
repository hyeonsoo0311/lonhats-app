# Feature Spec

## Workout

- Workout session date, memo, perceived difficulty.
- Sets with exercise, set count, reps, load, and minutes.
- Exercise library with target muscle, purpose, level, and key cue.

## Meal

- Meal entries with food name, amount, calories, protein, carbs, and fat.
- Future AI parser can convert natural language input into structured food entries.

## Journal

- Mood tag.
- One-line diary.
- Small win.
- Future letter or weekly reflection.

## Weekly Analysis

- Average calories in.
- Average active calories or workout output.
- Average workout minutes.
- Difference from daily target.
- Conservative recommendation for next week.

## Future Enhancements

- Supabase login and cloud sync.
- User profile: height, weight, goal weight, goal mode.
- Food database or external nutrition API.
- Progress photos and body measurements.
- Push reminders that do not shame the user.
