# Design Principles

## Visual Direction

The app should feel like a quiet training diary, not a cold fitness dashboard. Use warm paper, ink, moss green, tomato, sky, and amber accents. Avoid a one-color theme.

## UI Rules

- First screen is the usable dashboard, not a landing page.
- Cards are for repeated records, metrics, and focused input blocks only.
- No cards inside cards.
- Buttons use icons plus concise labels.
- Text must be compact on tool surfaces and larger only for the main dashboard headline.
- Empty states should feel calm, not apologetic.

## Voice

Good:

- "어제보다 1mm 나은 오늘"
- "완벽하지 않아도 루틴은 이어졌습니다."
- "오늘은 도망가지 않았다."

Avoid:

- Overly aggressive diet language.
- Fake medical confidence.
- Generic motivational slogans.
